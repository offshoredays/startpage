## 🧪 테스트 방법

### 1️⃣ **기본 드래그**

**작업**:
1. 아무 카테고리나 클릭하고 드래그 시작
2. **F12** 눌러서 콘솔 열기
3. 다른 카테고리 위로 이동

**확인 사항**:
- ✅ 드래그 중인 카드가 반투명 (40%)
- ✅ 마우스를 카드 위쪽으로 이동 → **파란 선이 카드 위에 나타남**
- ✅ 마우스를 카드 아래쪽으로 이동 → **파란 선이 카드 아래에 나타남**
- ✅ 파란 선이 **펄스 애니메이션**으로 깜빡임
- ✅ 콘솔에 "🎯 드래그 시작" 로그 표시

**예상 로그**:
```
🎯 드래그 시작: 여행
📍 드롭 위치: { from: "여행", to: "쇼핑", position: "위" }
✅ 새 순서: 여행 → 쇼핑 → 업무 → 개발
✅ 드래그 종료
```

---

### 2️⃣ **정확한 위치 삽입**

**초기 상태**:
```
[1. 여행]   [2. 쇼핑]
[3. 업무]   [4. 개발]
```

**작업**:
1. "여행" 드래그
2. "개발" 카드의 **상단**으로 이동
3. **파란 선이 "개발" 위에 나타나는지 확인**
4. 드롭

**결과**:
```
[2. 쇼핑]   [1. 여행]  ← 정확히 위에 삽입!
[3. 업무]   [4. 개발]
```

**콘솔 확인**:
```
📍 드롭 위치: { from: "여행", to: "개발", position: "위" }
✅ 새 순서: 쇼핑 → 업무 → 여행 → 개발
```

---

### 3️⃣ **자유 배치 테스트**

**목표**: 작은 카테고리를 왼쪽에, 큰 카테고리를 오른쪽에

**작업**:
1. 북마크 3개인 "여행" 카테고리 드래그
2. 맨 위 카테고리의 **위쪽**으로 이동
3. 파란 선 확인
4. 드롭
5. 북마크 5개인 "엔터" 카테고리도 왼쪽으로 이동

**예상 결과**:
```
[여행-3개]    [쇼핑-20개]
[엔터-5개]    [업무-15개]
              [개발-25개]
```
