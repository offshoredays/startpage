# 🚀 빠른 시작 가이드

## 1️⃣ GitHub 업로드 (3분)

### 필수 업로드 파일 (4개만!)
1. ✅ **README.md** (새로 작성됨)
2. ✅ **FINAL_CHECKLIST.md** (이 체크리스트)
3. ✅ **index.html** (CSS 링크 수정)
4. ✅ **js/widgets.js** (검색 위젯 복구)

### 방법
```
1. GitHub 저장소 접속
2. 각 파일 클릭 → ✏️ 편집
3. 전체 내용 붙여넣기
4. Commit (메시지: "v2.4.4 Final - 최적화 완료")
```

## 3️⃣ 서핑지수 API 설정 (5분, 선택사항)

### API 키 발급
1. https://www.data.go.kr 로그인
2. https://www.data.go.kr/data/15142490/openapi.do 접속
3. "활용신청" → 용도: "개인 스타트 페이지"
4. 승인 대기 (보통 즉시)
5. 마이페이지 → 개발계정 → **일반 인증키 (Encoding)** 복사

### 위젯 설정
1. 날씨 위젯 ⚙️ 클릭
2. API 키 입력
3. 해수욕장 선택 (울산/부산/강릉)
4. 표시 정보 선택
5. 저장

---

## 4️⃣ 클라우드 동기화 설정 (3분, 선택사항)

### GitHub Token 발급
1. https://github.com/settings/tokens/new 접속
2. `gist` 권한만 선택
3. 토큰 생성 → 복사

### 연결
1. 헤더 클릭 → 전체 설정
2. GitHub 클라우드 동기화 → 토큰 입력
3. "GitHub 연결하기" 클릭
4. ✅ 완료!
