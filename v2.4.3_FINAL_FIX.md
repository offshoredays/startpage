### 🔴 **Critical Bug 1: 주식/검색 위젯 설정 초기화**

**문제**:
- 서버에 파일 업로드 후 주식 위젯의 추가된 주식 사라짐
- 검색 위젯에 추가한 검색 엔진 사라짐
- 북마크는 정상 유지되지만 위젯 설정만 초기화됨

**원인**:
```javascript
// ❌ 문제 코드 (js/github-sync.js Line 335)
pullData() {
    // ... 클라우드 데이터 적용
    this.app.settings = cloudData.settings;
    
    // Re-render
    this.app.render();
    applyWidgetSizes(this.app);
    
    // ❌ 위젯 재초기화 누락!
    // 이전 설정으로 초기화된 위젯이 그대로 남음
}
```

**해결**:
```javascript
// ✅ 수정 코드
pullData() {
    // ... 클라우드 데이터 적용
    
    // 🔥 위젯 재초기화 추가
    console.log('🔄 Re-initializing widgets with restored settings...');
    initSearchWidget(this.app);
    initStockWidget(this.app);
    initWeatherWidget(this.app);
    console.log('✅ Widgets reinitialized');
}
```

**효과**:
- ✅ GitHub Gist에서 데이터 pull 후 위젯이 복원된 설정으로 재초기화
- ✅ 주식 심볼, 검색 엔진 리스트가 정상 복원
- ✅ 설정이 localStorage + GitHub Gist 양쪽에 모두 동기화

### 🔴 **Critical Bug 2: 카테고리 자유 배치 불가**

**문제**:
- CSS Column 레이아웃으로 인해 순서대로만 배치됨
- "왼쪽 2개, 오른쪽 4개" 같은 자유로운 배치 불가능
- 드래그 앤 드롭은 동작하지만 시각적 피드백 부족

**원인**:
```css
/* ❌ 이전 코드 (Column 레이아웃) */
.categories-wrapper {
    column-count: 2;
    column-gap: 16px;
}
/* ➡️ 순서대로만 채워짐 (왼→오, 위→아래) */
```

**해결**:
```css
/* ✅ 수정 코드 (Grid 레이아웃 + 자동 높이) */
.categories-wrapper {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    gap: 16px;
    grid-auto-rows: minmax(auto, auto); /* 자동 높이 */
    align-items: start; /* 위쪽 정렬 */
}

.category-card {
    height: fit-content; /* 내용에 맞춰 높이 */
    cursor: move; /* 드래그 힌트 */
}

.category-card:hover {
    box-shadow: var(--shadow-lg);
    border-color: var(--primary);
    transform: translateY(-2px);
}

.category-card.drag-over {
    background-color: rgba(59, 130, 246, 0.1);
    border: 2px dashed var(--primary);
    transform: scale(1.02);
}
```

**개선 사항**:
```javascript
// ✅ 드래그 오버 비주얼 피드백 추가
card.addEventListener('dragover', (e) => {
    e.preventDefault();
    card.classList.add('drag-over'); // 파란 테두리 표시
});

card.addEventListener('dragleave', (e) => {
    card.classList.remove('drag-over');
});
```

**효과**:
- ✅ Grid 레이아웃으로 전환 → 자유로운 순서 변경 가능
- ✅ 각 카테고리가 내용에 맞춰 자동 높이 조절 (Masonry 효과)
- ✅ 드래그 시 `cursor: move` + hover 효과로 직관적
- ✅ 드래그 오버 시 파란 테두리 표시 (명확한 피드백)

## 🧪 테스트 방법

### 1️⃣ **주식/검색 위젯 설정 복원 테스트**

**집 컴퓨터 (데이터 추가)**:
```javascript
// 1. 콘솔 클리어
console.clear();

// 2. 검색 엔진 추가
// Settings (⚙️) → Search Widget → Add search engine
// 예: "YouTube" / "https://www.youtube.com/results?search_query="

// 3. 주식 추가
// Settings (⚙️) → Stock Widget → Add stock
// 예: "AAPL" / "Apple Inc."

// 4. 현재 데이터 확인
console.log('검색 엔진:', Object.keys(app.settings.searchEngines || {}).length);
console.log('주식:', app.settings.stockSymbols?.length || 0);

// 5. 수동 push (즉시 동기화)
await app.githubSync.pushData();
console.log('✅ Pushed to cloud');
```

**회사 컴퓨터 (데이터 pull)**:
```javascript
// 1. 콘솔 클리어
console.clear();

// 2. 강제 새로고침 (Ctrl + Shift + R)

// 3. 수동 pull
await app.githubSync.pullData(true);

// 4. 검증
console.log('검색 엔진:', Object.keys(app.settings.searchEngines || {}).length);
console.log('주식:', app.settings.stockSymbols?.length || 0);
console.log('✅ Should match home computer data');

// 5. UI 확인
// Settings (⚙️) → Search Widget → 검색 엔진 리스트 확인
// Settings (⚙️) → Stock Widget → 주식 리스트 확인
```

**예상 결과**:
```
✅ 집 컴퓨터: 검색 엔진: 5, 주식: 3
✅ 회사 컴퓨터: 검색 엔진: 5, 주식: 3
✅ UI에서 모든 항목이 정상 표시됨
```

### 2️⃣ **카테고리 자유 배치 테스트**

1. **카테고리 생성**:
   - 카테고리 6개 생성 (다양한 북마크 수)
   - 예: 
     - 여행 (3개 북마크)
     - 쇼핑브랜드 (15개 북마크)
     - 업무 (8개 북마크)
     - 개발 (20개 북마크)
     - 엔터테인먼트 (5개 북마크)
     - 금융 (10개 북마크)

2. **드래그 테스트**:
   ```
   [여행]       [쇼핑브랜드]      →  [엔터테인먼트]  [여행]
   [업무]       [개발]           →  [금융]         [업무]
   [엔터]       [금융]           →  [쇼핑브랜드]    [개발]
   ```

3. **확인 사항**:
   - ✅ 카테고리 hover 시 `cursor: move` 표시
   - ✅ 드래그 시 카드 투명도 50% + scale 0.95
   - ✅ 드래그 오버 시 파란 테두리 표시
   - ✅ 드롭 후 순서가 즉시 변경됨
   - ✅ 각 카테고리가 내용에 맞춰 자동 높이

4. **자유 배치 테스트**:
   ```
   목표: 왼쪽에 작은 카테고리, 오른쪽에 큰 카테고리
   
   [여행 - 3개]         [쇼핑브랜드 - 15개]
   [엔터 - 5개]         [개발 - 20개]
                        [금융 - 10개]
   ```
