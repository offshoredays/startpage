## 🔍 근본 원인 분석

### 버그 1: 초기화 순서 문제
**Before:**
```javascript
async init() {
    if (this.githubSync.isConfigured()) {
        await this.githubSync.initialSync();  // 클라우드에서 데이터 불러옴
    }
    
    this.loadSettings();  // ❌ 로컬 설정으로 덮어씀!
    loadFooterBookmarks(this);  // ❌ 로컬 푸터로 덮어씀!
}
```

**After:**
```javascript
async init() {
    if (this.githubSync.isConfigured()) {
        const syncSuccess = await this.githubSync.initialSync();
        
        // Sync 실패했을 때만 로컬에서 로드
        if (!syncSuccess) {
            this.loadData();
            this.loadSettings();
            loadFooterBookmarks(this);
        }
        // Sync 성공하면 이미 pullData()에서 모든 데이터 로드됨
    } else {
        // GitHub 연결 안 됐을 때만 로컬 로드
        this.loadData();
        this.loadSettings();
        loadFooterBookmarks(this);
    }
}
```

### 버그 2: 설정 병합 문제
**Before:**
```javascript
if (cloudData.settings) {
    this.app.settings = { ...this.app.settings, ...cloudData.settings };
    // ❌ 기존 설정과 병합 → 새 설정이 제대로 적용 안 됨
}
```

**After:**
```javascript
if (cloudData.settings) {
    console.log('📥 설정 적용:', Object.keys(cloudData.settings).length, '항목');
    this.app.settings = cloudData.settings;  // ✅ 완전 덮어쓰기
}
```

### 버그 3: localStorage 백업 불완전
**Before:**
```javascript
// pullData() 에서
this.app.saveToLocalStorage();  // 카테고리만 저장
```

**After:**
```javascript
// pullData() 에서
this.app.saveToLocalStorage();  // 카테고리 저장
localStorage.setItem('settings', JSON.stringify(this.app.settings));  // 설정 저장
localStorage.setItem('footerBookmarks', JSON.stringify(this.app.footerBookmarks));  // 푸터 저장
console.log('💾 로컬 백업 저장 완료');
```

```javascript
console.log('=== 현재 상태 ===');
```

**Gist ID가 다르면 아래 실행!**

### 3단계: Gist ID 통일 (다를 경우만)

**회사 컴퓨터** (최신 데이터):
```javascript
// Gist ID 확인 및 강제 업로드
console.log('Gist ID:', localStorage.getItem('github_gist_id'));
await app.githubSync.pushData();
```

**집 컴퓨터**:
```javascript
// 회사 컴퓨터의 Gist ID를 여기에 붙여넣기

// 강제 다운로드
await app.githubSync.pullData(true);

// 확인
console.log('북마크:', app.categories.reduce((s,c)=>s+c.bookmarks.length,0));
```
