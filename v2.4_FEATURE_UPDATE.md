## 🧪 테스트 방법

### 1. 카테고리 크기 확인
```
1. 페이지 강제 새로고침 (Ctrl+Shift+R)
2. 북마크가 적은 "여행" 카테고리 확인
3. 북마크가 많은 "쇼핑브랜드" 카테고리와 크기 비교
4. "여행"이 더 작게 표시되어야 함!
```

### 2. 서핑지수 (API 키 없이도 작동!)
```
1. 페이지 새로고침
2. 날씨 위젯 확인
3. 파도 높이 옆에 "🏄 보통" 같은 표시가 있어야 함!
4. 색상도 함께 표시됨
```

### 3. 서핑 API 키 설정 (선택사항)
```
1. 설정 (⚙️) 열기
2. "서핑지수 API 설정" 섹션 찾기
3. [API 키 발급하기] 링크 클릭
   → https://www.data.go.kr/data/15097079/openapi.do
4. 발급받은 API 키 입력
5. 저장
6. 날씨 위젯에서 실시간 서핑지수 확인!
```

## 📊 서핑지수 API 정보

### API 제공
- **기관**: 해양수산부 국립해양조사원
- **이름**: 서핑지수 API
- **URL**: https://www.data.go.kr/data/15097079/openapi.do
- **비용**: 무료
- **제공 정보**: 파도 높이, 파도 방향, 서핑지수

### 울산 해역 코드
```javascript
const obsCode = 'DT_0042'; // 울산 관측소 코드
```
*실제 코드는 API 문서에서 확인 필요*

### API 요청 예시
```javascript
const apiUrl = `http://www.khoa.go.kr/api/oceangrid/surfForecast/search.do` +
    `?ServiceKey=${surfApiKey}` +
    `&ObsCode=DT_0042` +
    `&Date=20260309` +
    `&ResultType=json`;
```

### 응답 구조 (예상)
```json
{
  "result": {
    "data": [
      {
        "wave_height": 1.2,
        "wave_dir": "NE",
        "surf_idx": "좋음"
      }
    ]
  }
}
```

## 💡 꿀팁

### 1. API 키 없이도 사용 가능!
```javascript
// 바람 속도 기반 서핑 조건 추정
파도 높이 (m) ≈ 0.3 × 풍속 (m/s)

예시:
- 풍속 2m/s → 파도 0.6m → 서핑지수 "보통"
- 풍속 5m/s → 파도 1.5m → 서핑지수 "좋음"
```

### 2. 서핑 최적 조건
```
매우좋음: 파도 1.5~2.5m
좋음: 파도 1.0~1.5m
보통: 파도 0.5~1.0m
낮음: 파도 0.5m 이하
위험: 파도 2.5m 이상
```

### 3. 실시간 확인
```
날씨 위젯 → 10분마다 자동 업데이트
서핑지수도 함께 갱신!
```
